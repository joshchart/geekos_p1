# Tutor Templates

Clean starting versions of session files. Use these to reset for a new student or project.

## Files

| Template | Copies To | Purpose |
|----------|-----------|---------|
| `student-preferences.md` | `session/student-preferences.md` | Learning profile and background |
| `code-review-checklist.md` | `session/code-review-checklist.md` | Coding standards developed in sessions |
| `project-progress.md` | `session/projectN/progress.md` | Component completion status |
| `project-experiences.md` | `session/projectN/experiences.md` | Learning journal |
| `nextSteps.md` | `session/nextSteps.md` | Project status and next steps |

## How to Reset

See `instructor/reset-instructions.md` for detailed reset commands for:
- New student starting fresh
- Same student starting a new project
- Restarting a project from scratch
